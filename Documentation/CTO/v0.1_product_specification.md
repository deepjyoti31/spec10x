## 1. Product Overview

### What It Does

Spec10x v0.1 lets product managers upload raw customer interviews (transcripts, audio, video) and get AI-synthesized insights ‚Äî themes, pain points, feature requests, sentiment ‚Äî without any manual tagging, organizing, or pre-categorization.

### Core Principles

| Principle | What It Means |
|---|---|
| **Zero-config start** | Upload files ‚Üí get value. No setup wizards, no project creation |
| **AI organizes, not the user** | No projects, no folders. The AI discovers themes across ALL data |
| **Trust through transparency** | Every AI insight links to exact source quotes. Users can verify and correct |
| **Progressive disclosure** | Top-level dashboard is simple; depth is one click away |
| **Speed** | Everything feels instant ‚Äî streaming responses, optimistic updates |

### What the User Gets

1. **Automatic theme discovery** ‚Äî AI reads all interviews and clusters them into themes ranked by urgency and frequency
2. **Evidence trails** ‚Äî Every theme links to exact quotes from exact interviews
3. **Sentiment analysis** ‚Äî Know if feedback is positive, negative, or mixed
4. **Natural language Q&A** ‚Äî Ask questions about your interviews in plain English (or any language)
5. **Export** ‚Äî Get insights as structured markdown or PDF

## 2. User Journeys

### 2.1 New User ‚Äî First-Time Flow

```
Sign Up (email/Google) 
    ‚Üí Land on Insight Dashboard (empty state)
    ‚Üí Click "Upload Interviews" 
    ‚Üí Upload Modal opens
    ‚Üí Drag & drop files
    ‚Üí AI processes (transcription + analysis)
    ‚Üí Metadata review step (optional, skippable)
    ‚Üí Close modal ‚Üí Dashboard populates with AI-discovered themes
    ‚Üí Explore themes, read quotes, drill into interviews
    ‚Üí Try "Ask Your Interviews" to query the data
    ‚Üí Export insights as markdown/PDF
```

**Key moments:**
- **First wow:** Seeing themes auto-appear in the dashboard after uploading raw files
- **Trust moment:** Clicking a theme and seeing the exact quotes that support it
- **Power moment:** Asking a question in the Ask view and getting a cited answer

### 2.2 Returning User ‚Äî Ongoing Flow

```
Log In 
    ‚Üí Land on Insight Dashboard (populated)
    ‚Üí See "New" badges on themes that emerged since last visit
    ‚Üí Review new/updated themes
    ‚Üí Upload more interviews (+ button in sidebar)
    ‚Üí Dashboard auto-updates ‚Äî themes evolve, new ones appear
    ‚Üí Use Ask view for specific queries
    ‚Üí Export updated insights
```

### 2.3 Upload Flow (detailed)

```
Click "+ Upload" (sidebar or empty state)
    ‚Üí Upload Modal opens (large, overlay on dashboard)
    ‚Üí Drag & drop files into drop zone (or click to browse)
    ‚Üí Files appear in processing queue:
        Queued ‚Üí Transcribing (audio/video) ‚Üí Analyzing ‚Üí Done
    ‚Üí Live insight preview shows themes streaming in as files complete
    ‚Üí All files processed ‚Üí Metadata Review step appears:
        Each file listed by name with its detected/missing metadata:
        "interview_sarah.mp3 ‚Äî We detected 2 speakers"
        "feedback_notes.pdf ‚Äî No speakers detected"
        [Speaker 1: ___] [Role: ___] [Company: ___]
        [Skip ‚Äî it's okay to leave these blank]
    ‚Üí User fills in or skips
    ‚Üí "Done ‚Äî View Insights" button ‚Üí Close modal ‚Üí Dashboard updated
```

### 2.4 Exploration Flow

```
Dashboard shows top 10 themes (sorted by urgency √ó frequency)
    ‚Üí Click a theme card ‚Üí Right panel shows full detail:
        All quotes, sentiment breakdown, affected segments, related themes
    ‚Üí Click "View full interview ‚Üí" on a quote ‚Üí Navigate to Interview Detail View
    ‚Üí Read full transcript with color-coded AI highlights
    ‚Üí Correct/adjust any AI extractions that seem wrong
    ‚Üí Navigate back to Dashboard (breadcrumb)
    ‚Üí Click "Show more themes" to see beyond top 10
    ‚Üí Rename a theme via right-click or edit icon
```

### 2.5 Ask Flow

```
Click "Ask ‚ú®" in top nav ‚Üí Ask Your Interviews page opens
    ‚Üí First-time: see suggested starter questions
    ‚Üí Type a question: "What are the biggest pain points with onboarding?"
    ‚Üí AI responds with structured answer + inline citation badges
    ‚Üí Click a citation badge ‚Üí jumps to that interview
    ‚Üí See suggested follow-up questions below the response
    ‚Üí Continue conversation with follow-up queries
```

## 6. Edge Cases & Rules

| Scenario | Behavior |
|---|---|
| **Duplicate file uploaded** | SHA-256 hash check ‚Üí warn: "This file appears to already exist. Upload anyway?" |
| **Non-interview content** | Accept and process. If few/no insights found: flag as "Low insight content" in the sidebar |
| **Video/audio exceeds duration limit** | Block processing. Show plan limit + suggest uploading transcript instead |
| **Video processing** | Extract audio ‚Üí speech-to-text ‚Üí discard video. Only store transcript |
| **Contradictory feedback** | Show sentiment split on theme card. Highlight as "Polarizing theme" if split is close to 50/50 |
| **Sarcasm/misinterpretation** | AI flags low-confidence extractions with üö©. User can manually correct |
| **Context collapse** | Quote detail always shows surrounding lines for context via "Show context" expand |
| **Theme with only 1 source** | Labeled as "Signal", hidden by default, visible with "Show all" |
| **Theme with 2+ sources** | Full "Theme" status, shown as a card |
| **Old themes (no new mentions)** | Preserved in "Previous themes" collapsible section with muted styling |
| **New themes (since last visit)** | "NEW" badge on theme card, auto-removed after user clicks the theme |
| **Delete interview** | Confirmation: "Deleting this will remove X insights and may affect Y themes." Recalculates themes |
| **Multilingual interviews** | Process natively. Present insights in user's interface language. Quotes in original language + "Translate" button |
| **Multi-speaker audio/video** | Speaker diarization ‚Üí Speaker 1, 2, 3‚Ä¶ User assigns names in metadata review |
| **Free tier limit reached (10 interviews)** | Upload button shows upgrade prompt. Cannot upload more until upgrading or deleting existing |
| **Theme rename** | Inline edit on theme card. Change reflected everywhere that theme is referenced |
| **Sample data** | "Try with sample data" loads a pre-built demo dataset so new users can explore before uploading |
